# Presentaciones

Presentaciones generadas con rmarkdown, beamer

## Instalación 

    apt install r-cran-rmarkdown texlive-latex-extra texlive-lang-spanish texlive-fonts-extra

O instalación completa:

    apt install r-cran-rmarkdown texlive-full
